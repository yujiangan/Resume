★ PersonaHub — AI 驱动的 GitHub 用户画像分析工具 AI
https://github.com/yujiangan/-PersonaHub
技术栈： TypeScript React Nitro ai-sdk SSE

项目背景：

输入任意 GitHub 用户 ID，基于 LLM + ReAct 智能体整合用户公开数据，经多轮分析 + 流式报告生成，实时呈现分析过程与 Markdown 报告，面向 HR 筛选、候选人评估、社区初识等场景

核心工作：

  • 独立完成 GitHub 用户画像分析工具从需求梳理、方案设计到前后端落地，实现用户输入账号后自动生成可读画像报告，并支持实时展示分析过程。

技术亮点：

  • LLM Agent 全链路落地：围绕用户画像场景设计“数据理解 → 多源分析 → 报告生成”的智能体流程，将分散的 GitHub 公开信息转化为结构化候选人画像。
  • 实时分析体验：将长耗时分析拆解为可感知的流式反馈，让用户在等待过程中看到当前进度、分析思路与最终报告，降低工具使用的不确定感。
  • 稳定性与容错设计：针对外部数据不完整、请求失败、限流和超时等情况设计兜底策略，避免单个数据源异常导致整体分析中断。
  • 工程化交付：完成前后端类型约束、部署配置与线上访问链路，仓库语言占比 90.1% TypeScript。
